Purpose
=======
    These files contain sample java codes with java classes and
interfaces that illustrate the application of generic types with 
lambda expressions.

     The value of this sample code is to show various scenarios 
that generic types and anonymous classes and functional interfaces
and lambda expressions can work together.

Requirements
============
. Java >= 1.8 (OpenJDK and Oracle JVMS have been tested)


Getting started
===============

1. javac -Xlint *.java (to view all the warnings as well)

2. java <One of the Class names>
