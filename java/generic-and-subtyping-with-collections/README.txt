Purpose
=======
    These files contain sample java codes with java classes and
interfaces that illustrate the application of generic subtyping 
in conjunction with Java Collections.

    The value of this programming pattern lies in its ability to 
allow developers to write generic methods for code reuse.

===============

1. javac -Xlint *.java (to view all the warnings as well)

2. java <One of the Class names>
