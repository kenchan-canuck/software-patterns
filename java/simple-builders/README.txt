Purpose
=======
    These files contain sample java codes with java classes and
interfaces that illustrate the application of simple inheritance
and nested inner class to implement the Builder Pattern.

    The value of these sample codes is to demonstrate the simplicity
of using basic java linguistic features such as Inheritance and 
static nested inner class to implement the Builder Pattern.

Getting started
===============

1. javac -Xlint *.java (to view all the warnings as well)

2. java <One of the Class names>
